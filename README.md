# sosty-media
Sosty Media è un hub multimediale leggero per i monitor di corridoio delle scuole. Sviluppato in Python con FastAPI e SQLite, include un pannello stile WordPress per scrivere news istituzionali, caricare .mp4 o immagini e integrare YouTube o Drive. Supporta l'ordinamento Drag&amp;Drop con SortableJS e l'avvio automatico con audio.
